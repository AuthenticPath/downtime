Downtime App Demo

How It Works

Welcome to the Downtime App Demo simulation. Here’s how it works:

  • Each user has a contacts list and personalized tier assignments which determine connection preferences.
  • You can toggle user availability to control who can join chats.
  • Start a 1‑on‑1 or group chat using the chat buttons. Only available users are considered.
  • You can invite users to a chat, add new users to your contacts, or remove users from the network.
  • Your tier settings and contacts determine which users can connect with one another.

Enjoy exploring the simulation!
